CSS là viết tắt của Cascading Style Sheets (phong cách biểu định dải tầng)
CSS tiết kiệm được rất nhiều công việc, nó có thể kiểm soát bố cục của nhiều trang web cùng lúc.

Ta đang sử dụng cách CSS internal
Còn 1 cách sử dụng CSS nữa là CSS external.

CSS external được sử dụng để định kiểu cho nhiều trang web khác nhau
Sử dụng tag <link> để liên kết đến file CSS bên ngoài.
Để sử dụng cách CSS external trong trang này, hãy bỏ comment của <link> trong phần <head> của trang web đi và cho phần <style>...</style> thành commemt

  1. HTML Link - thuộc tính mục tiêu
    1. Thông thường, trang liên kết sẽ được hiển thị ngay trong cửa sổ trình duyệt hiện tại. Để thay đổi điều đó, ta phải chỉ định mục tiêu khác cho liên kết
      Thuộc tính <target> chỉ định nơi để mở tài liệu liên kết.
    2. Thuộc tính <target> có thể có 1 trong các giá trị sau:
      • _self - Mặc định của thuộc tính <target>. Mở tài liệu ở trong 1 cửa sổ/tag khi nó được nhấp vào.
      • _blank - Mở tài liệu trong 1 tag mới hoặc của sổ mới.
      • _parent - Mở tài liệu trong khung chính.
      • _top - Mở tài liệu trong toàn bộ phần thân của cửa sổ.
      + Ví dụ:
      • _self - Click vào đây để xem cách chạy của _self.
      • _blank - Click vào đây để xem cách chạy của _blank.
      • _parent - Click vào đây để xem cách chạy của _parent.
      • _top - Click vào đây để xem cách chạy của _top.

        <style>
            a:link {
                color: green;
                background-color: transparent;
                text-decoration: none;
            }
                    
            a:visited {
                color: pink;
                background-color: transparent;
                text-decoration: none;
            }
                    
            a:hover {
                color: red;
                background-color: transparent;
                text-decoration: underline;
            }
                    
            a:active {
                color: yellow;
                background-color: transparent;
                text-decoration: underline;
            }
        </style>
