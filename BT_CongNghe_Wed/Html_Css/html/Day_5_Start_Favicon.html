HTML Favicon (biểu tượng HTML)

Là 1 hình ảnh nhỏ, đơn giản bất kì, có độ tương phản cao.
Được hiển thị ở bên trái tag. Cú pháp để chạy biểu tượng (favicon) là:

<link rel="icon" type="image/x-icon" href="/images/favicon.ico">

Chú ý: Thêm nó vào phần <head> sau phần <title>

E) Bảng có sọc vằn

Để biến cái bảng của bạn có các 2 màu xen kẽ nhau (giống vằn) ta sử dụng thuộc tính :nth-child(even/odd). Code mẫu:
tr:nth-child(odd){
background-color:lemonchiffon;
}
Có thể thay tr thành th hoặc td tùy ý.

Chú ý: Đoạn code mẫu chỉ có thể viết bằng CSS và nằm trong style_day5.css
