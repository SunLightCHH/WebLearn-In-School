Bài 5. CSS Căn bản - Hình ảnh nền
CSS3 cho phép thêm nhiều hình nền cho trang Web thông qua thuộc tính background-image. Tạo trang Web với 2 hình nền, hình thứ nhất được canh lề = dưới bên phải, hình thứ 2 được canh lền trên bên trái.
Lưu ý: Mỗi hình khác nhau được đặt cách nhau bằng dấu phầy, độ ưu tên theo thứ tự.
